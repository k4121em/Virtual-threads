# Mockup Animasi 3D

Aplikasi ini mengubah desain 2D menjadi animasi 3D dengan efek angin, animasi berjalan, dan ekspor video yang lancar.

## Fitur Utama
- Unggah gambar 2D.
- Pilih efek animasi (angin, gerakan, dll.).
- Pratinjau animasi 3D.
- Ekspor video.

## Cara Menggunakan
1. Unggah gambar 2D.
2. Pilih efek animasi.
3. Pratinjau animasi.
4. Ekspor video.
